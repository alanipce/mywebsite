Projects

Red Folder Initiative
Red Folder is a reference tool that contains safety tips and contact information for a wide variety of emergency campus resources. It also provides guidance on responding to incidents in which a community member might report being a danger to self or a threat to others.
Student Advising
Student Advising was my first project as a Mobile App Developer, it was web based hybrid app for the Student Advising department. It included advising resources and utilities necessary for any student to plan out a successful college experience.
